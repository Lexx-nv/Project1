Документ "Счет на оплату поставщика"

Данный документ относится к подсистеме "Управление автотранспортом".

Реквизиты шапки документа

  • Организация. Организация, от лица которой выставлен счет за услуги предоставления транспорта и строительной техники.
  • Банковский счет. Банковский счет этой организации, на который будут переведены средства.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • Контрагент. Заказчик услуг транспорта и строительной техники.
  • Договор контрагента. Документ - основание платежа.
  • Объект строительства. Реквизит необязателен для заполнения. Если он заполнен, то в счет будет выставлен только за работу транспорта и строительной техники на данном объекте.
  • Сумма документа. Реквизит рассчитывается автоматически при записи документа.

Заполнение табличной части документа

Табличная часть может быть по желанию пользователя заполнена автоматически за произвольно выбранный период времени по кнопке "Заполнить". В табличную часть попадают данные  из регистра накопления "Выработка по стоимости", аккумулируемые там при проведении Товарно-транспортных документов, относящихся к данному Заказчику и данному Объекту Строительства (если заполнен реквизит Объект строительства ). Детальность заполнения табличной части (сумма услуг по объекту в целом, либо количество услуг по каждому параметру выработки, либо по каждой модели транспортных средств, либо по каждому транспортному средству в отдельности) определяется настройкой Учетной политики Управления автотранспортом (закладка Прейскуранты, реквизит Детализация выработки для клиента).Чтобы получить максимально подробный отчет по оказанным за данный период услугам заказчику, необходимо сформировать отчет "Реестр на выписку счета".

Ввод на основании

Документ может быть введен на основании документа "Поступление товаров услуг".

Проведение документа

Документ не производит движений по регистрам.
