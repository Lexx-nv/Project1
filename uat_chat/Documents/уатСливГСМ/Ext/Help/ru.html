Документ "Слив ГСМ"

Назначение

Данный документ служит для регистрации операций по сливу ГСМ из бака автомобиля на склад.

Относится к подсистеме "Учет ГСМ".

Параметры

Длина номера: 5, тип номера: Число.

Периодичность номера: Непериодический

Тип номера: Число

Реквизиты

  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство или оборудование.
  • ГСМ. Ссылается на справочник «Номенклатура».
  • Склад. Ссылается на справочник «Склады». Склад.
  • Количество. Тип значения: «Число». Краткое описание.
  • Путевой лист. Ссылается на документ «Путевой лист». 

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "ГСМ --> Слив ГСМ"

Движения

  • «Остатки ГСМ в ТС». Отражаются партии остатков ГСМ на ТС. Документ выполняет расход по данному регистру.
  • «Партии товаров на складах». Отражаются партии остатков товаров (материалов) на складах. Документ выполняет приход по данному регистру.
