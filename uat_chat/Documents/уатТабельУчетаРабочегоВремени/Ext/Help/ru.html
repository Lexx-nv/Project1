Документ "Табель учета рабочего времени"

Предназначен для составления организациями "Табелей учета рабочего времени" в соответствии с Постановлением Госкомстата России от 5 января 2004 г. Позволяет сформировать табель по указанной организации и подразделению организации.

Данный документ относится к подсистеме "Управление персоналом".

Реквизиты шапки документа

  • Организация. Организация, к которой относятся сотрудники.
  • Подразделение. Подразделение организации, к которому относятся сотрудники.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • Комментарий. Дополнительная произвольная текстовая информация.

Табличная часть "Основная таблица"

Содержит таблицу использования сотрудниками рабочего времени по дням месяца.

Кнопка "Заполнить" позволяет сформировать таблицу по имеющейся в базе информации об использовании рабочего времени:

  • При проведении путевых листов, формируется отработанное время каждого сотрудника, с учетом времени простоя и ночного времени.
  • При проведении ремонтных листов, в соответствии с заполненной табличной частью закладки "Исполнители", формируется время каждого сотрудника в ремонте.
  • По данным документов Корректировка использования рабочего времени.
  • Для переходящих путевых листов, у которых не указана дата возвращения, в табель попадает плановое время работы, заданное в реквизите "Продолжительность рабочего дня" в настройке учетной политики.

Таблица "Корректировка использования рабочего времени"

Предназначена для быстрого ввода по указанным сотрудникам корректировки рабочего времени. Например, это может быть необходимо при вводе больничных дней или отпуска. При этом создается один документ "Корректировка рабочего времени".

Форма "Настройка"

Предназначена для настройки формирования табеля. Вызывается при нажатии кнопки "Настройка" командной панели основной таблицы, а также при нажатии на гиперссылки "Начисления" и "Неявки". Имеет следующие реквизиты:

  • Начисления 1-8. Вид рабочего времени, используемый при расчете итогов для начислений зарплаты сотрудникам.
  • Неявки 1-8. Вид рабочего времени, используемый для учета отклонений.
  • Перезаполнять таблицу табеля при изменении настроек и корректировке. При установке флага таблица табеля и итоги по начислениям и неявкам будут автоматически пересчитываться при изменении настроек или вводе корректировок рабочего времени.

Способ вызова

Вызывается из меню "Управление персоналом --> Учет рабочего времени --> Табель учета рабочего времени сотрудников".

Движения

Документ не производит движений по регистрам.

Дополнительная информация

Кнопка "Печать / Список невыясненных дней" позволяет получить список только тех водителей, у которых в табеле остались пустые графы. Кнопка "Печать / Бланк" позволяет получить пустые бланки табелей со списком работников, но без данных об использовании рабочего времени.

Ночное отработанное время не отображается в таблице табеля, но используется в начислениях зарплаты (это время отображается в печатной форме Т-13, если в настройках установлено использование ночного времени в начислениях).

Внимание! Для того, чтобы при проведении путевых и ремонтных листов, информация по отработанному времени использовалась в табеле, в обработке Установка прав и настроек должен быть установлен флажок "Учитывать рабочее время водителей для формирования табеля"!
