Документ "Выгрузка регламентированных отчетов"

Назначение

Документ содержит первичную информацию по выгрузкам регламентированных отчетов. Документ может содержат несколько отчетов, например, сформированных за квартал по какой либо организации. Работа с объектами регламентированной отчетности удобна в обработке "Общие объекты регламентированной отчетности".

Относится к подсистеме "Регламентированная отчетность".

Параметры

Длина номера: 13.

Периодичность номера: Год.

Тип номера: Строка.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой вводится выгрузка. 
  • Код ИФНС. Тип значения: «Строка». Код ИФНС (инспекции федеральной налоговой службы).
  • Комментарий.  Тип значения: «Строка». Произвольный комментарий. 

Табличная часть «Основная»

  • Основание. Ссылается на документ «Регламентированный отчет». Регламентированный отчет по которому строится выгрузка.
  • Номер строки текста выгрузки. Тип значения: «Число». Номер строки выгрузки. 

Табличная часть «Выгрузки»

  • Тип. Ссылается на перечисление «Версии форматов выгрузки». Версия формата выгрузки.
  • Имя файла. Тип значения: «Строка». Полное имя файла выгрузки.
  • Текст. Тип значения: «Строка». Текст выгрузки отчета. 

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма навигации по ошибкам. Дополнительная форма. Используется для удобного представления ошибок, возникших в результате формирования выгрузки.

Способ вызова

Вызывается из основной формы управления отчетностью "Общие объекты регламентированной отчетности".

Движения

  • «Архив данных регламентированной отчетности». Содержит для хранения данных по регламентным отчетам.
