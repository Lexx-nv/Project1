Документ «Инвентаризация агрегатов»

Назначение

Документ «Инвентаризация агрегатов» предназначен для проведения инвентаризации агрегатов. Есть два вида проведения инвентаризации: на складах, на ТС.

При проведении инвентаризации на складах данные в инвентаризационной ведомости можно автоматически заполнить информацией о наличии агрегатов на указанном в документе складе с помощью кнопки «Заполнить».

При проведении инвентаризации на ТС данные в инвентаризационной ведомости можно автоматически заполнить информацией об установленных на ТС агрегатах с помощью кнопки «Заполнить».

Данные о наличии агрегатов отображаются в графе «Наличие по данным учета» и не редактируются. В графе «Наличие» указывается реальное наличие агрегата на складе/ТС, полученное в результате проведенной инвентаризации.

Проведение документа не производит никакого движения в регистрах, однако, по результатам инвентаризации можно выписать подчиненные документы: "Операции с агрегатами", "Поступление агрегатов" и "Списание агрегатов".

Относится к подсистеме "Эксплуатация".

Параметры

Длина номера: 11, тип номера: Число.

Периодичность номера: Периодический, в пределах года.

Тип номера: Строка

Реквизиты

  • Организация. Ссылается на справочник "Организации". Организация, где проводится инвентаризация.
  • Склад. Ссылается на справочник "Склады". Склад организации.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "Складской учет --> Инвентаризация агрегатов".

Движения

Отсутствуют.
