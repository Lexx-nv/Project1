Документ "Распределение затрат"

Документ предназначен для распределения затрат между транспортными средствами. Распределение затрат выполняется за период с начала по конец месяца даты документа. Данным документом можно отразить следующие расходы следующего характера:

  • Производственные;
  • Косвенные;

Заполнение и смысл реквизитов

При вводе документа в шапке необходимо указать следующие реквизиты.

  • "Организация" - указать организацию, для транспортных средств которой будет выполняться распределение затрат.
  • "Колонна - указать колонну, для которой будет выполняться распределение затрат. Если колонна не указана, то распределение затрат будут выполняться для всех транспортных средств организации.
  • "Распределять затраты по пут. листам" - если флажок включен, до по всем проведенным путевым листам будет распределены затраты на ГСМ и износ шин.
  • "Распределять затраты на ремонт" - если флажок включен, до по всем проведенным ремонтным листам будет распределены затраты на ремонт и техническое обслуживание.

Табличные части:

"Косвенные затраты"

В это табличной части заполняется перечень косвенных затрат, подлежащих распределению. Сумма распределения указывается в валюте регламентированного учета.

"Износ ТС"

В это табличной части заполняется список транспортных средств и указывается их износ. Таблица может заполняться автоматически на основании справочника "Транспортные средства". Сумма износа указывается в валюте регламентированного учета.

Способ вызова

Вызывается из меню "Затраты --> Распределение затрат".

Движения

  • «Затраты ТС». Выполняются движения по распределению производственных и косвенных затрат.
