Документ "Реализация услуг"

Документ предназначен для отражения факта оказания услуг.

Относится к подсистеме "Сервис".

Реквизиты шапки документа

  • Организация. Организация, от имени которой оформляется документ;
  • Контрагент. Получатель услуг;
  • Договор контрагента. Договор по которому оказываются услуги;
  • Ответственный. Пользователь ответственный за документ.

Заполнение табличной части документа

Табличная часть может быть по желанию пользователя заполнена автоматически за произвольно выбранный период времени или еще не выставленными услугами по кнопке "Заполнить". В табличную часть попадают данные  из регистра накопления накапливаемые при проведении «Товарно-транспортных документов», относящихся к данному заказчику. Детальность заполнения табличной части (сумма услуг по объекту в целом, либо количество услуг по каждому параметру выработки, либо по каждой модели транспортных средств, либо по каждому транспортному средству в отдельности) определяется настройкой Учетной политики Управления автотранспортом (закладка Прейскуранты, реквизит Детализация выработки для клиента).

Способ вызова

Вызывается из меню "Работа ТС --> Услуги --> Реализация услуг"
