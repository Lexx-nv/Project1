Документ "Поступление материалов и услуг"

Документ служит для отражения различных операций по поступлению товаров.   

Если для данного документа были сформированы соответствующие документы типовых конфигураций 1C, то такой документ уже нельзя изменить. Для возможности внесения изменений необходимо удалить сформированные типовые документы. Это может сделать только пользователь с правами «Администратор управления автотранспортом» из меню «Сервис»-«Регламентные операции УАТ»-«Список сформированных документов типовой конфигурации для УАТ». Необходимо найти нужный документ УАТ и нажать клавишу Del. Все сформированы для него документы типовой конфигурации будут помечены на удаление.

Относится к подсистеме "Складской учет".

Ввод на основании

Документ может быть введен на основании документа "Счет на оплату поставщика".

Способ вызова

Вызывается из меню "Складской учет --> Поступление материалов и услуг"
