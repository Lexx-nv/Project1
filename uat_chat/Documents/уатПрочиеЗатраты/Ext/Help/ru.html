Документ "Прочие затраты"

Документ предназначен для отражения в расходах нематериальных затрат, стоимость которых известна по реквизитам первичного документа и не подлежит корректировке при закрытии месяца. Данным документом можно отразить следующие расходы следующего характера:

  • Производственные;
  • Брак в производстве;
  • Общепроизводственные (ОПР);
  • Общехозяйственные (ОХР);
  • Коммерческие (КР);
  • Вложения во внеоборотные активы.

Особенности

В БУ отражаемая операция соответствует списанию на затраты с кредита непроизводственных счетов.

Заполнение и смысл реквизитов

При вводе документа в шапке необходимо указать следующие реквизиты
Реквизиты шапки:

  • "Заказ" - указать аналитику затрат "Заказ". Распределение косвенных затрат (ОПР, ОХР, КР) если для них задан заказ, выполняется только на выпуск продукции по заданному заказу.
  • "Подразделение организации" - подразделение организации (юридического лица), на которое списываются затраты.

В табличной части необходимо указать список затрат, при этом заполняются следующие реквизиты:

  • "Статья затрат" - аналитика затрат. В строке после выбора статьи затрат выводится вид расходов для этой статьи затрат.
  •  "Номенклатурная группа" - аналитика затрат, указывает, под выпуск какой номенклатурной группы списывается сумма на затраты. Распределение косвенных затрат (ОПР, ОХР, КР) если для них задана номенклатурная группа, выполняется только на выпуск продукции заданной группы.
  • "Счет" - кредит счета проводок затрат. Дополнительно при необходимости следует указать аналитику счета.
  • "Сумма" - сумма затрат в валюте регламентированного учета.

Движения

  • «Затраты ТС». Выполняются движения по прочим затратам.
