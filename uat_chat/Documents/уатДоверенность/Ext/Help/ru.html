Документ "Доверенность"

Назначение

Документ "Доверенность" предназначен для оформления доверенностей на получение товаров. В документ из справочника "Сотрудники" вносятся данные о доверенном лице, путем выбора информации о собственных сотрудниках компании из справочника. При этом будут автоматически заполнены паспортные данные сотрудника. Табличная часть доверенности заполняется путем подбора или построчного ввода.

Относится к подсистеме "Складской учет".

Параметры

Длина номера: 11.

Периодичность номера: Год.

Тип номера: Строка

Реквизиты

  • Сотрудник. Ссылается на справочник «Сотрудники (УАТ)». Подотчетное лицо.
  • Организация. Ссылается на справочник «Организации». Организация, по которой вводится документ.
  • Банковский счет организации. Ссылается на справочник «Банковские счета». Банковский счет организации.
  • Контрагент. Ссылается на справочник «Контрагенты». Поставщик товаров (материалов).
  • Договор контрагента. Ссылается на справочник «Договоры контрагентов». Договор контрагента.
  • Сделка. Ссылается на документы «Поступление товаров и услуг», «Реализация услуг». Документ расчетов, по которому выписывается доверенность. Поле может оставаться пустым.

Табличная часть «Товары»

  • Наименование товара. Тип значения: «Строка». Представление номенклатурной позиции.
  • Количество. Тип значения: «Число». Количество получаемых товаров.
  • Единица измерения. Ссылается на справочник «Классификатор единиц измерения». Единица измерения.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "Складской учет --> Доверенность".
