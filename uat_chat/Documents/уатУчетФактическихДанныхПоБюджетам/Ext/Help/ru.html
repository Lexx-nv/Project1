Документ «Учет фактических данных по бюджетам»

Документ «Учет фактических данных» предназначен для ввода в подсистему бюджетирования данных о фактической деятельности предприятия, структурированных в соответствии с аналитическими измерениями бюджетирования.

Существуют три существенных отличия в хранении в подсистеме бюджетирования плановых и фактических данных:

  • для фактических данных не указывается сценарий планирования; они используются для анализа и сравнения со всеми остальными сценариями;
  • учет фактических данных в подсистеме производится по дате их возникновения, а плановые обороты учитываются датой начала периода планирования;
  • фактические данные всегда хранятся с периодичностью в день; при проведении план-фактного анализа они  группируются по периодичности планирования сценария, по которому проводится выборка плановых данных в анализе.

Ввод фактических данных возможен как в ручном, так и в автоматическом режиме. В последнем варианте, в шапке документа указываются фильтры по статьям оборотам и ЦФО, а также период отбора фактических данных. В этом случае система проведет отбор всех статей оборотов согласно настроенным фильтрам и сформирует фактические обороты.

Правила отбора фактических данных для каждой статьи оборота настраиваются в форме элемента справочника «Статьи оборотов по бюджетам» на закладке «Данные». После автоматического заполнения табличная часть может быть скорректирована пользователем вручную.

Важно: Независимо от порядка определения границ отбора данных, установленных для источников, используемых в данном случае для получения фактических данных по статьям оборотов, выборка по всем источникам будет производиться по указанному в документе интервалу.

Изменение остатков по плану счетов бюджетирования производится при проведении документа автоматически на основании шаблонов проводок, указанных для статей оборотов.
