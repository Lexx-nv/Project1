Отчет «Контролируемые значения по бюджетам»

Отчет предназначен для получения текущего состояния оборотов по комбинациям аналитических измерений бюджетирования, для которых установлены контролируемые значения (ограничивающие или целевые).

Отчет формируется для конкретного сценария, на определенный период, определяемый датой его начала и периодичностью планирования по сценарию. Порядок настройки отчета в целом аналогичен другим отчетам.

На основной форме указываются:

  • сценарий, для которого анализируются контролируемые значения;
  • вид анализируемых контролируемых значений (ограничивающее или целевое);
  • способ использования контролируемых значений (при планировании или исполнении планов);
  • дата начала периода, для которого анализируются контролируемые значения;
  • отборы по ЦФО и проектам (при необходимости, реквизиты вызываются по кнопке «Отбор» главного меню).

На дополнительной форме настройки, вызываемой по кнопке «Настройка» главного меню основной формы, указываются те же самые реквизиты, что и на основной. Также, дополнительно настраивается:

  • состав показателей, выводимых в отчет;
  • вид отчета (размещение измерений отчета в строках и столбцах);
  • отборы;
  • признак вывода показателей отчета в разных колонках.

В состав измерений отчета по умолчанию включаются:

  • статья оборотов;
  • контролирующий сценарий;
  • начало контролируемого периода: это значение особенно важно, когда обороты контролируются по оборотам другого сценария с большей периодичностью, чем контролируемый.

Например. Если обороты сценария с периодичностью в месяц контролируются по сценарию с периодичностью в квартал, датой начала контролируемого периода будет дата начала квартала, в который попадает месяц, выбранный для контроля. При этом в отчете будут анализироваться именно суммарные обороты за квартал,  в который попадает период отчета.
