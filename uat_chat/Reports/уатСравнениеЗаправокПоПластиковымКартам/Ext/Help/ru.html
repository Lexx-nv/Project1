Отчет "Сравнение заправок по пластиковым картам"

Назначение

Предназначен для сравнения фактических заправок ГСМ по документам "Заправка ГСМ" и данных, предоставляемых поставщиками ПЦ по документам "Отчет поставщика ПЦ".

Показатели расхода ГСМ имеют такое количество знаков после запятой, какое указано в настройках учетной политики организации.

В случае, если пластиковая карта не найдена в базе данных, но присутствует в документе "Отчет поставщика ПЦ", то она будет помечена красным цветом.

Относится к подсистеме "Управление Автотранспортом".

Способ вызова

Вызывается из меню "ГСМ -> Сравнение заправок по пластиковым картам"

Настройка отчета

Отчет строится с обязательными отборами по Организации. Вариант сравнения влияет на то, какие данные будут учитываться

По кнопке «Сформировать» выполняется построение отчета в соответствии с текущей настройкой.

Кнопки сохранения и восстановления значений позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.
