Регистр накопления "Заказы на ТС"

Назначение

В этом регистре хранятся данные по сделанным заказам на ТС. Заказы открываются документами «Заказ на ТС» и могут быть выполнены документами видов «Путевой лист», «Маршрутный лист», или закрыты документами «Закрытие заказов на ТС»,.

Остаточный.

Относится к подсистеме: «Работа ТС».

Измерения

  • Заказчик. Тип значения: справочник «Подразделения организаций», справочник «Контрагенты». Заказчиком может выступать контрагент или подразделение (внутренний заказ). 
  • Заказ на ТС. Тип значения: документ «Заказ на ТС». Заказ на ТС. 
  • Номенклатура. Тип значения: справочник «Номенклатура». Заказанная номенклатура (груз). 
  • Единица измерения. Тип значения: справочник «Единица измерения», справочник «Классификатор единиц измерения». Единица измерения груза. 
  • Дата выполнения. Тип значения: «Дата». Дата выполнения заказа. 
  • Приоритет. Тип значения: «Число». Приоритет выполнения заказа перевозки груза.
  • Параметр выработки. Тип значения: справочник «Параметры выработки». Параметр выработки.

Ресурсы

  • Количество. Тип значения: «Число». Количество перевозимого груза.
  • Количество (по параметру выработки). Тип значения: «Число». Количество затраченных единиц измерения (время, расстояние и т.п.) согласно параметру выработки на перевозку груза.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заказ на ТС», «Закрытие заказов на ТС», «Маршрутный лист», «Путевой лист».
