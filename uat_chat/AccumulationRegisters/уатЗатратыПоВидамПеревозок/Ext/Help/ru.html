Регистр накопления "Затраты по видам перевозок"

Назначение

В этом регистре хранятся данные по распределенным затратам на ТС по видам перевозок.

Оборотный.

Относится к подсистеме: «Управление затратами».

Измерения

  • Вид перевозки. Тип значения: справочник «Виды перевозок». Вид перевозки, по которому ведется учет затрат.
  • Статья затрат. Тип значения: справочник «Статьи затрат». Статья затрат распределения. 

Ресурсы

  • Сумма. Тип значения: «Число». Сумма в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Путевой лист».
