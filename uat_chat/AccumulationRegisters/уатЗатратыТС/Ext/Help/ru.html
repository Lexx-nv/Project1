Регистр накопления "Затраты ТС"

Назначение

В этом регистре хранятся данные по распределенным затратам на ТС.

Оборотный.

Относится к подсистеме: «Управление затратами».

Измерения

  • Контрагент. Тип значения: справочник «Контрагенты». Контрагент, по которому ведется учет затрат.
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет затрат.
  • Подразделение. Тип значения: справочник справочник «Подразделения организаций». Подразделение, по которому ведется учет затрат.
  • Транспортное средство. Тип значения: справочник «Транспортные средства». Транспортное средство, по которому ведется учет затрат.
  • Параметр выработки. Тип значения: справочник «Параметры выработки». Параметр выработки. Необходимая аналитика.
  • Заказ на ТС. Тип значения: документ «Заказ на ТС».
  • Документ выпуска. Тип значения: документ «Ввод остатков ГСМ», документ «Ввод начальных данных по агрегатам», документ «Путевой лист», документ «Распределение затрат», документ «Выдача расходных материалов», документ «Ремонтный лист». Документ, на который распределяются затраты. Например, затраты на ремонт - ремонтный лист, затраты на зарплату - путевой лист и т.п.
  • Статья затрат. Тип значения: справочник «Статьи затрат». Статья затрат распределения. 
  • Счет затрат. Тип значения: план счетов «Регламентированный». Счет затрат. 
  • Затрата. Тип значения: справочник «Серии номенклатуры», справочник «Номенклатура», справочник «Транспортные средства». Номенклатура затрат или транспортное средство.

Ресурсы

  • Количество. Тип значения: «Число». Единицы измерения количества определяются параметром выработки: с, км.
  • Сумма. Тип значения: «Число». Сумма в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Прочие затраты», «Распределение затрат».
