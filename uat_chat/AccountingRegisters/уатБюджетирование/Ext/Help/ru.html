Регистр бухгалтерии «Бюджетирование (уат)»

Регистр бухгалтерии «Бюджетирование (уат)» используется для хранения плановых и фактических данных подсистемы бюджетирование управления автотранспортом.

Регистр использует план счетов «Бюджетирование (уат)» и поддерживает корреспонденции. Это означает, что каждая запись регистра имеет дебетовую и кредитовую часть. Такая организация регистра позволяет получать информацию не только об остатках и оборотах по счетам, но и о корреспонденциях между счетами.

Измерения

Сценарий

Балансовое измерение регистра «Сценарий» обеспечивает возможность хранить одновременно несколько вариантов плановых, а также фактические данные за один и тот же период. Фактические данные отличаются от плановых тем, что это измерение для них не заполняется.

Валюта

Измерение регистра «Валюта» является небалансовым и обеспечивает учет по валютным счетам в разрезе отдельных наименований иностранных валют.

Ресурсы

Сумма упр.

Балансовый ресурс «Сумма упр.» обеспечивает отражение как плановых, так и фактических данных в валюте управленческого учета на всех счетах плана счетов. Это относится и к предопределенным счетам и к тем, которые введены в режиме 1С:Предприятие.

Валютная сумма

Ресурс «Валютная сумма» является небалансовым, а значит, может вестись выборочно, для отдельных счетов. Такие счета имеют признак «Валютный». Ведение валютного учета по каждому виду субконто на таких счетах настраивается отдельно.

Ведение валютного учета по счету в целом и по отдельным видам субконто дает возможность получать остатки и обороты в валюте в разрезе валют.

Сумма сценария

Ресурс «Сумма сценария» отображает данные о плановых хозяйственных операциях в валюте, выбранной для сценария планирования. Ресурс заполняется только для плановых данных.

Количество

Ресурс «Количество» является небалансовым и ведется только для тех счетов, для которых имеет смысл учет в натуральных измерителях. Такие счета имеют признак «Количественный». Ведение количественного учета по каждому виду субконто на таких счетах настраивается отдельно.

Реквизиты

Статья оборотов

Реквизит содержит ссылку на элемент справочника «Статьи оборотов по бюджетам», выбранный при описании хозяйственной операции.
