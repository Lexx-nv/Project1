Обработка "Установить цену заправок"

Обработка предназначена для установки новой цены топлива в документах "Заправка ГСМ".

При открытии фомы обработки необходимо выбрать период формирования , организацию, АЗС и уазать новую цену. Затем включить флажки для тех видов заправок, по которым требуется установить новую цену и  нажать кнопку "Выполнить".

После установки цен и перепроведении документов "Заправка ГСМ", данные движений путевых листов становятся некорректными. Для сохранения корректности данных движений ГСМ выполняется восстановление последовательности, которое может занять довольно продолжительное время. Поэтому рекомендуется использовать стандартный механизм восстановления последовательностей (Главное меню Операции -> Проведение документов -> Восстановление последовательностей).
