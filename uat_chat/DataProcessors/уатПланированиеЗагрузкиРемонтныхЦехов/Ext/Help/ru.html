Обработка "Планирование загрузки ремонтных цехов"

Назначение

Обработка предназначена для планирования загрузки ремонтных цехов на основе «Заявок на ремонт». На диаграмме в графическом виде отображается планируемая загрузка ремонтных цехов по времени и указываются пересечения работ в одном цехе.

Относится к подсистеме: «Управление ремонтами».

Формы

  • Форма. Позволяет задать параметры вывода диаграммы: период анализа, цеха, отборы и представление выводимых документов.
  • Форма диаграммы. Позволяет просмотреть результаты планирования загрузки ремонтных цехов.
  • Форма настройки печати. Позволяет просмотреть и настроить детализацию диаграммы перед выводом на печать. Для настройки детализации используется период.

Способ вызова

Вызывается из меню "Ремонты --> Планирование загрузки цехов".

Дополнительная информация

Данные обработки представлены в виде диаграммы Ганта. Может формироваться как только для цехов, так и с детализацией по документам "Заявка на ремонт". Цеха отображаются в виде иерархии, в том виде, в котором они введены в справочнике. При раскраске интервалов учитывается планируемый период ремонта по заявке и график работы цеха. Отображается загрузка только для нижнего уровня цехов, верхние уровни иерархии цехов отображаются, но загрузка по ним не выводится.

Информационные поля документов

  • Состояние - состояние заявки на ремонт.
  • Модель - Модель транспортного средства, указанного в заявке.
  • ТС - Транспортное средство, указанное в заявке.
  • Мастер - Главный мастер цеха, указанный в заявке.
  • Причина обращения - Причина обращения, указанная в заявке.

Цвета раскраски интервалов диаграммы Ганта:

  •           - не рабочее время (по графику работы цеха);
  •           - планируется занять (по заявке на ремонт);
  •           - планируется занять (по цеху в целом);
  •           - имеются пересечения занятости (по одному цеху), то есть несколько заявок введено по одному цеху на один период работ.
