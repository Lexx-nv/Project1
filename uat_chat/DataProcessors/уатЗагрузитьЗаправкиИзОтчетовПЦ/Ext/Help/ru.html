Обработка "Загрузить заправки из документов ПЦ"

Обработка предназначена для загрузки заправок из документов "Отчет поставщика ПЦ (процессингового центра)". При загрузки формируются документы "Заправка ГСМ".

 

При открытии фомы обработки необходимо выбрать период формирования , организацию и АЗС. Затем включить флажки для тех видов ГСМ, по которым требуется выполнить загрузку и нажать кнопку "Выполнить".

 

Установленный флажок "Загрузить только проверенные" означает, что из документов "Отчет поставщика (процессингового центра)" будут загружаться будут только строки с установленным флажком "Проверено".

 

Если установить флажок "Обновить путевые листы", то при формировании для каждой заправки будет выполняться поиск путевого листа по автомобилю и дате заправки. В случае нахождения такого путевого листа, заправка будет в него добавлена и по путевому листу будет пересчитан расход топлива.
