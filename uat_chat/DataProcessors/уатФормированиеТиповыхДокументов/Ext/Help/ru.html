Обработка "Формирование типовых документов"

Назначение

Эта обработка предназначена для автоматического формирования документов типовых конфигураций 1С: УПП, БП, УТ.

Относится к подсистеме "Обмен данными".

Правила создания документов:

  • На основании проведенных документов «Заправка ГСМ» формируются типовые документы различного вида. Вид создаваемого документа зависит от вида заправки:
    • Для видов заправки «Со склада» и «По талонам» будет создан стандартный документ «Требование-накладная»,
    • Для вида заправки «За наличные» будет создан стандартный документ «Авансовый отчет»,
    • Для вида заправки «По пластиковой карте» и «От поставщика» будет создан стандартный документ «Поступление товаров и услуг».
  • На основании проведенных документов «Путевой лист» формируются документы «Выработка ОС». При этом из путевых листов переносится информация о тех параметрах выработки (справочник «Параметры выработки ТС и водителей»), у которых выбрано значение реквизита «Параметр выработки ОС».
  • На основании проведенных документов «Ремонтный лист» формируются документы видов:
    • Для ремонта в собственной ремонтной зоне будет создан стандартный документ «Требование-накладная», 
    • Для ремонта в автосервисе будет создан стандартный документ «Поступление товаров и услуг».
  • На основании проведенных документов «Выдача расходных материалов» будет создан стандартный документ «Требование-накладная».
  • На основании проведенных документов «Товарно-транспортный документ» формируются документы «Реализация товаров и услуг».
  • На основании складских документов формируются одноименные стандартные документы.
  • На основании документа «Начисление зарплаты» формируется стандартный документ «Начисление зарплаты работникам организаций».
  • В колонке "Значение" дерева настроек на первой странице по строкам формирования общих документов можно ввести Дату общего документа. Общий документ - это вариант выгрузки всех документов, когда документы одного вида перегружаются в один стандартный документ.
  • Также в колонке "Значение" напротив некоторых документов можно указать Контрагента, при этом будет использован отбор по этому значению. Если значение не заполнено, то отбор по контрагенту не используется.

Способ вызова

Вызывается из меню "Регламентные операции --> Формирование документов типовых конфигураций 1С".
