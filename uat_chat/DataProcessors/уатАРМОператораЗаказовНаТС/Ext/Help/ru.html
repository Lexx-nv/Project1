Обработка "АРМ оператора заказов на ТС"

Обработка "АРМ Оператора заказов на ТС " предназначена для автоматизации рабочего места оператора. Основное назначение обработки -  помощь в регистрации заказов на транспортные средства.  

Вся информация функционально сгруппирована по закладкам.

 Закладка " Новый заказ".

 Закладка предназначена для для регистрации новых заказов на ТС. Кнопка "Создать" формирует  документы, заполняя их данными данной страницы

 Закладка " Список заказов".

 Закладка отображает журнал документов "Заказ на ТС"".

 Закладка " Список заявок ". (доступна только при совместном использовании данного решения с бухгалтерией предприятия или с управлением торговлей)

 Закладка предназначена для формирования документов "Заказ на ТС" на основании документов "Заказ покупателя" или "Реализация товаров и услуг" типовых конфигураций. Тип документа на основании которого будут формироваться Заказы на ТС указывается в учетной политике. Закладка содержит списки текущих, обработанных и отклоненных заявок. Для формирования заказов на ТС необходимо указать параметры отправления и отметить типовые документы из списка для которых будет производиться формирование. В списке возможно изменить некоторые параметры типовых документов. В списке обработанных заявок отображается соответствие документов заявок и сформированных на их основании Заказов на ТС.
