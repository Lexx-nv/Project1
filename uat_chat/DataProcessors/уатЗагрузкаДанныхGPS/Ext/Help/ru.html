Обработка "Загрузка данных системы навигации (GPS)"

Назначение

Эта обработка позволяет производить загрузку данных местоположения ТС по системе спутниковой навигации (GPS) с использованием файла обмена. 

Относится к подсистеме: «GPS».

Формы

  • Форма. Основная форма. На форме расположены следующие элементы:
    1. Поля основных настроек Период (будут загружены только те строки из файла, период которых находится в заданном диапазоне), Разделитель (служебный символ, использующийся для разделения полей в пределах одной строки), Имя файла обмена (текстовый файл выгрузки данных из системы GPS с определенным форматом формирования строк/полей).
    2. Дополнительные настройки - Порядок полей в файле (настраивается последовательность полей в строке файла для корректной синхронизации с регистром при загрузке данных) и Отбор по ТС (гибкая настройка по загружаемым ТС, если отбор не установлен, то будут загружены все данные. Например, можно указать загрузку ТС в определенном диапазоне гаражных номеров, или только для конкретной модели/типа ТС и т.п.)
    3. Таблица загружаемых данных - промежуточный результат загрузки данных из файла. Сначала данные из файла попадают в эту таблицу. Пользователь может их просмотреть, принять их и нажав кнопку "Выполнить" загрузить в базу, либо отклонить и не загружать данные в базу.

После закрытия формы (и при последующем открытии) сохраняются значения реквизитов: Файл обмена, Разделитель, таблица Поля файла обмена. Для автосохранения необходимо воспользоваться стандартными кнопками "Сохранение/восстановление значений".

Способ вызова

Вызывается из меню "Работа ТС --> Мониторинг --> Загрузка данных GPS".
