Обработка "Изменение норм расхода топлива"

Обработка предназначена для пакетного изменения норм расхода топлива.

При открытии фомы обработки необходимо выбрать дату изменения и, если необходимо, перейти на закладку "Отбор" и настроить необходимые параметры.

Закладка "Основная"

Необходимо указать величину изменения нормативного расхода ГСМ. Изменение может быть абсолютным, т.е. вместо старого значения линейной нормы будет установлено новое, или относительным - новое значение будет скорректировано на заданный процент.

При нажатии на кнопку "Заполнить" выводится список транспортных средств в соответствии с отбором установленном на закладке "Отбор".

Для установки новых значений необходимо нажать кнопку "Выполнить".

Закладка "Отбор"

"Модель"  - необязательный параметр. Устанавливает отбор транспортных средств по модели транспортных средств.

"Вид топлива"  - необязательный параметр. Устанавливает отбор транспортных средств по виду топлива.

"Тип ТС"  - необязательный параметр. Устанавливает отбор транспортных средств по типу транспортных средств.
