Задача "Задача исполнителя"

Назначение

Задачи являются средством коммуникации между пользователями, а также используются для автоматизации бизнес-процессов.   

Для создания новой задачи можно воспользоваться бизнес-процессом Согласование. Для создания задач используйте подменю "Создать на основании - Согласование". При этом объект-основание станет предметом задачи.

Для удобства просмотра можно установить отбор по автору, по исполнителю, просматривать только выполненные, невыполненные или все задачи, а также указать интервал дат по дате записи задачи.

Способ вызова

Вызывается из меню "Сервис --> Согласование документов (УАТ) --> Задачи"
