Регистр сведений "Картинки и файлы"

Назначение

Является универсальным хранилищем дополнительных данных (внешних объектов - файлов, картинок и т.п.).

 

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Объект. Ссылается на справочники: «Номенклатура», «Контрагенты», «Сотрудники», «Организации». Объект (справочник) для которого необходимо реализовать хранение некоторых данных (например картинок или электронных образов).
  • Идентификатор. Идентификационная строка, разделяющая различные данные объекта. Например: Картинка1, Картинка2,...

Ресурсы

  • Имя файла. Строковое значение. Исходное имя файла для сохранения.
  • Дата сохранения. Дата сохранения данных объекта.
  • Размер файла. Исходный размер файла.
  • Комментарий. Произвольный строковый комментарий к файлу / картинке.
  • Данные. Хранилище данных картинки или файла.
  • Данные во внешнем файле. Признак того, что файл хранится не в базе данных, а в качестве внешнего файла на диске.

Способ вызова

Вызывается из форм списков и форм элементов справочников, перечисленных в измерении "Объект" по кнопке "Перейти -> Картинки и файлы".
