Регистр сведений "Местоположение по GPS"

Назначение

Регистр предназначен для хранения истории текущего местоположения транспортных средств согласно системе спутниковой навигации (GPS).

Относится к подсистеме: «Работа ТС».

Параметры

Периодичность: Периодический, в пределах секунды.

Режим записи: Независимый.

Способ вызова

Вызывается из меню "Работа ТС --> Мониторинг --> Местоположение по GPS".
