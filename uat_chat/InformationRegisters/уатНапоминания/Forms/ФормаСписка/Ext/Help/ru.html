Форма списка регистра сведений "Напоминания"

Назначение

Форма предназначена для вывода списка напоминаний.

При просмотре списка напоминаний, можно отображать только те события, которые еще не произошли, устанавливать фильтр по определенному пользователю - получателю напоминания.

Форма имеет следующие элементы:

  • Кому - установка фильтра по имени адресата;
  • Показывать только незавершенные - флажок, отметка о выводе только незавершенных напоминаний;
  • Список элементов - список напоминаний.
