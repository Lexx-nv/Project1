Регистр сведений "Дополнительные сведения по Omnicomm"

Назначение

Регистр предназначен для хранения дополнительной информации из системы Omnicomm. Содержит записи о событиях заправок ГСМ, слива ГСМ, мгновенного превышения скорости, долгосрочного превышения скорости, нарушения предельных значений датчиков.

Относится к подсистеме "GPS".

Параметры

Периодичность: Периодический, в пределах секунды.

Режим записи: Независимый.

Способ вызова

Вызывается из меню "GPS --> Дополнительные сведения по Omnicomm".
