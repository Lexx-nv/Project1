Подсистема "Управление персоналом"

Назначение

Подсистема предназначена для ведения учета сведений по сотрудникам и водителям, учета рабочего времени водителей по параметрам выработки и начисления зарплаты сотрудникам.

Состав

  • Справочники
    • Графики работы
    • Классификатор использования рабочего времени
    • Смены
    • Тарифные планы начисления зарплаты водителям
    • Тарифы зарплаты водителей
  • Документы
    • Корректировка использования водителями рабочего времени
    • Начисление зарплаты
    • Путевой лист
    • Табель учета рабочего времени
  • Отчеты
    • Сводная ведомость по начисленной зарплате
    • Ведомость работы водителей
    • Рейтинг водителей по ДТП
  • Планы видов расчета
    • Основные начисления
  • Регистры сведений

    • Графики работы по видам времени
    • Регистрационные документы водителей

    • Тарифы водителя
