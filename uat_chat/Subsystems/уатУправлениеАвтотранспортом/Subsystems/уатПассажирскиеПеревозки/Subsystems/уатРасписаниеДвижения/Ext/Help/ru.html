Подсистема "Расписание движения"

Назначение

Подсистема предназначена для храния расписания выходов на маршруты для более точного учета перемещений пассажирских ТС.

Состав

  • Справочники
    • Выходы на маршруты
    • Рейсы маршрутов
  • Обработки
    • Расписание маршрутов
  • Регистры сведений
    • Расписание рейсов
