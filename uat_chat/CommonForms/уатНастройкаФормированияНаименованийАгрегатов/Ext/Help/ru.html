Форма "Настройка формирования наименований агрегатов"

Данная форма позволяет задать правило, по которому будет формироваться наименование агрегатов в системе.

Необходимо указать порядок следования реквизитов, по которому будет формироваться наименование.

Есть возможность выбора реквизитов агрегата. Можно произвольно добавить любую строку, которая тоже появится в наименовании.

Реквизит [ГоденДо] не будет подставляться в наименование в случае, если тип агрегата шина.

В случае, если никаких настроек не указать, то наименование будет формироваться по умолчанию.
