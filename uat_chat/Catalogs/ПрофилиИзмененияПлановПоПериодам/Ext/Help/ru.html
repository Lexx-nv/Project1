Справочник «Профили изменения планов по периодам»

Справочник используется для определения порядка формирования серии бюджетных транзакций в нескольких временных периодах на основании исходного значения и таблицы, указывающей, в каком периоде и с каким коэффициентом к исходному значению будут формироваться обороты.

Основные способы использования этого справочника следующие:

  1. Распределение исходной суммы оборотов по времени, например, планируемая сумма поступлений ДС от  реализации текущего месяца распределяется на текущий и два последующих месяца в пропорции 50-30-20
  2. Построение графика изменения значения какого - либо оборота по периодам (например, ежемесячное приращение на 10%)

Для настройки профиля необходимо указать шаг изменения данных во времени и заполнить таблицу «Профиль изменения», в которой для каждого формируемого оборота  указать смещение периода, в котором он будет сформирован, от момента возникновения исходного значения (другая бюджетная транзакция или расчет по модели), а также указать коэффициент, на который будет умножено исходное значение для определения величины оборота, приходящегося на данный период.
