Справочник "Планы применения затрат"

Назначение

Справочник предназначен для настройки правил применения затрат и аналитики.

Относится к подсистеме "Работа ТС".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой устанавливается план затрат.

Табличная часть «Затраты»

Табличная часть предназначена для настройки списка затрат, которые будут применяться в данном плане.

Табличная часть «Фильтр»

Табличная часть предназначена для настройки фильтров действия данного плана. В качестве элементов фильтра могут указываться значения колонн и транспортных средств.

Формы

  • Форма выбора. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Затраты --> Планы применения затрат".
