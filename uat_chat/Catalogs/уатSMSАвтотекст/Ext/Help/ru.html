Справочник "Автотекст"

Назначение

Справочник "Автотекст" предназначен для замены определенных представлений в теле собщения SMS на значения результат которых описан в формате внутреннего языка 1С Предприятия.

 

Реквизиты

  • Представление. Тип значения: «Строка».  Текст для поиска и замены. 
  • Автотекст. Тип значения: «Строка». Текст в формате внутреннего языка 1С Предприятия который выполняется при нахождении "Представления" в теле сообщения SMS.

 

Пример использования

  • Представление - '[Дата]'
  • Автотекст - 'Автотекст = Формат(ТекущаяДата(),"ДФ=dd.MM.yyyy")'

В данном случае при нахождении в тексте SMS строки '[Дата]' (без ковычек) она будет заменена на текущую дату в формате дд.мм.гггг
