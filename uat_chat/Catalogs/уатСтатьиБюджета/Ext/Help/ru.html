Справочник «Статьи бюджета»

Справочник «Статьи бюджета» подчинен справочнику «Бюджеты» и предназначен для отбора оборотов по статьям, которые будут включены в тот или иной бюджет. 

Справочник иерархический, статьи оборотов можно объединять в группы с расчетом итога по группе. Для каждого элемента справочника указывается:

  • статья оборотов, значение оборота по которой будет выведено в строку бюджета;
  • знак, с которым будет выведено и войдет в итог по группе значение оборота.
