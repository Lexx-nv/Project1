Справочник «Статьи отчета о прибылях и убытках»

Подчинен элементу справочника «Варианты настроек для финансового анализа» и содержит правила отбора оборотов по регистру «Обороты бюджетов»  для каждой статьи отчета о прибылях и убытках. Заполняется автоматически при первом открытии справочника по новому владельцу. В дальнейшем состав справочника не редактируется. В форме элемента справочника пользователь должен выбрать статьи (группы статей) из справочника «Статьи оборотов по бюджетам», обороты по которым войдут в данную статью отчета о прибылях и убытках, а также указать для каждой из выбранных статей (групп статей) оборотов знак, с которым обороты войдут в общий итог по статье отчета. При необходимости, пользователь может изменить имя статьи, выводимое в отчет. Подчиненным этому справочнику является справочник «Детализация отчета о прибылях и убытках», в котором для каждой статьи можно выбрать произвольное количество подстатей, которые также будут выведены в отчет. Настройка элемента справочника детализации аналогична настройке элемента справочника статей отчета о прибылях и убытках.
