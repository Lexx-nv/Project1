Справочник "Учетные записи электронной почты"

Назначение

Справочник предназначен для использования во встроенном почтовом клиенте и содержит настройки учетных записей электронной почты. Пользователь регистрирует необходимые учетные записи в справочнике. При этом назначается ответственный за учетную запись, пользователь, который может изменять ее параметры, назначать уровень доступа другим пользователям.

Относится к подсистеме "CRM".

Реквизиты

  • Параметры доставки (адрес, сервер, порт, логин, пароль и т.п.). Тип значения: «Строка», «Булево», «Число». Параметры подключения к почтовому серверу, авторизация, параметры приема/отправки писем.
  • Группы писем. Ссылается на справочник «Группы писем электронной почты». Пользователь может создать необходимые группы писем электронной почты у каждой учетной записи. Некоторые из них можно назначить в качестве предопределенных.
  • Параметры новых писем по умолчанию. Тип значения: «Строка», «Булево», «Число». Формат текста по умолчанию, текст подписи, флаги добавления подписи.
  • Параметры автоматической обработки писем. Тип значения: «Строка», «Булево», «Число». Указываются такие параметры, как Интервал автополучения писем, действие при автополучении, автосохранение писем и прочие.

Табличная часть «Доступ к учетной записи»

Настройка доступа к учетной записи. Должны быть указаны пользователи, имеющие доступ к учетной записи, и некоторые роли доступа.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. 

Способ вызова

Вызывается из меню "CRM --> Электронная почти --> Учетные записи электронной почты".

Особенности работы со справочником

На закладке «Подключение» указываются стандартные параметры для подключения к почтовому серверу.

Можно организовать автополучение/отправку электронных писем, при этом необходимо назначить ответственного пользователя, у которого будет выполняться данная операция, и выбрать тип операции (получение, отправка, отправка и получение). Просмотреть и обновить параметры автополучения/отправки электронных писем текущего пользователя можно в соответствующем пункте меню «Действия» главной консоли встроенного почтового клиента.

Для каждой учетной записи можно организовать учет писем по предметам. При этом имеется возможность для новых писем заполнять предмет из темы письма, или по теме письма определять предмет из уже существующих писем в ИБ (для входящих).

Можно указать список пользователей, которым доступна работа с письмами данной учетной записи на различные действия:

  • Чтение. Доступно только чтение писем
  • Отправка. Доступно чтение, редактирование и создание новых писем
  • Транспорт. Доступна только операция отправки и получения писем, без возможности создания новых.

Существует возможность удалять помеченные на удаление электронные письма без установки монопольного режима (закладка «Дополнительно»). 

Для учетной записи можно настроить фильтры, которые будут работать при получении новых электронных писем.
