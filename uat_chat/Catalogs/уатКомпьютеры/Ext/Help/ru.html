Справочник "Компьютеры"

Назначение

Содержит список всех компьютеров локальной сети, с которых осуществляется доступ к информационной базе. При первом запуске с нового рабочего места, автоматически создается новый элемент справочника.

Относится к подсистеме "Сервис".

Параметры

Длина кода: 27, тип кода: Число.

Длина наименования: 30.

Не иерархический.

Реквизиты

  • Локальный каталог. Тип значения «Строка». Каталог в который копируются компоненты и настройки системы защиты, для ускорения запуска информационной базы и возможности работы распределенной информационной базы в отрыве от сети.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.

Способ вызова

Вызывается из меню "Основные данные --> Предприятие --> Компьютеры".
