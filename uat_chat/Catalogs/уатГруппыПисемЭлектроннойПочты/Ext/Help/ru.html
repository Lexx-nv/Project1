Справочник "Группы писем электронной почты"

Назначение

Справочник предназначен для использования во встроенном почтовом клиенте конфигурации и является подчиненным справочнику "Учетные записи электронной почты". Все электронные письма системы можно хранить по группам в каждой учетной записи. Наглядное отображение групп электронных писем показано на форме основной консоли встроенного почтового клиента.

Относится к подсистеме "CRM".

Параметры

Длина кода: 5, тип кода: Строка.

Длина наименования: 25.

Не иерархический.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
