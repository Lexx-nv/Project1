Справочник "Колонны ТС"

Назначение

Справочник содержит список колонн (транспортных подразделений). Нахождение ТС в составе колонны носит периодический характер, определяется в регистре сведений «Местонахождение ТС».

Относится к подсистеме "Работа ТС".

Реквизиты

  • Параметр выработки для распределения затрат. Ссылается на справочник «Параметры выработки». Используется для распределения затрат по колоннам (документ «Распределение затрат»).
  • Вести учет затрат. Тип значения «Булево». Флаг ведения затрат по колонне.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.

Способ вызова

Вызывается из меню "Основные данные --> Транспортные средства, оборудование --> Колонны транспортных средств".
