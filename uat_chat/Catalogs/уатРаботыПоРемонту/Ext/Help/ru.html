Справочник "Работы по ремонту ТС"

Назначение

В справочник авторабот заносится информация обо всех видах работ по ремонту ТС, выполняемых на предприятии, включаемых в документы "Ремонтный лист".

Относится к подсистеме "Управление ремонтами".

Параметры

Длина кода: 13, тип кода: Строка.

Длина наименования: 100.

Иерархический: группы и элементы.

Реквизиты

  • Артикул (№ по каталогу). Тип значения: «Строка». Номер автоработы по каталогу.
  • Наименование полное. Тип значения: «Строка». Полное наименование автоработы.
  • Номенклатура. Ссылается на справочник «Номенклатура». Базовая номенклатура автоработы. В случаях, когда планируется выгружать документы «Ремонтный лист» в типовое решение "Бухгалтерия предприятия", данный реквизит рекомендуется заполнять.

Способ вызова

Вызывается из меню "Ремонты --> Работы по ремонту".
