Справочник "Параметры выработки (ТС и водителей)"

Назначение

Справочник содержит список параметров выработки, по которым производится расчет в путевых листах. Справочник содержит перечень предопределенных элементов.

Для новых элементов возможно задавать произвольные формулы расчета параметров.

Относится к подсистеме "Работа ТС".

Реквизиты

  • Параметр выработки ОС. Ссылается на справочник «Параметры выработки ОС». Используется при заполнении документа "Выработка ОС".
  • Временной. Признак того, что параметр выработки является временным, т.е его значение измеряется в часах.
  • Коэфициент. Тип значения: «Число». Используется для пересчета выработки при заполнении документа "Выработка ОС". 
  • Действует на ТС. Тип значения: «Булево». Флаг действия на ТС.
  • Действует на сотрудников. Тип значения: «Булево». Флаг действия на сотрудников/водителей.
  • Использовать при планировании. Тип значения: «Булево». Флаг использования при планировании.
  • Использовать в тарифах на услуги. Тип значения: «Булево». Флаг использования в тарифах на услуги при расчете стоимости услуг.
  • Весовой коэффициент затрат. Тип значения: «Число». Используется при распределении затрат на ТС. 0 - затраты не распределяются, 1 - полное распределение затрат.
  • Наименование для документов. Тип значения: «Строка». Используется для заполнения счетов и актов.
  • Способ расчета параметра выработки. Ссылается на перечисление «Способы расчета параметров выработки». Используется для ограничения реквизитов, по которым будет рассчитываться параметр выработки.
  • Формула расчета. Тип значения: «Строка». Используется для расчета произвольных параметров выработки.

Способ вызова

Вызывается из меню "Основные данные --> Классификаторы --> Параметры выработки транспортных средств и сотрудников".
