Справочник "Значения свойств объектов"

Назначение

Предназначен для хранения значений дополнительных свойств различных объектов, физический смысл которых не отражен в других справочниках конфигурации. В справочник "Значения свойств" вводятся подчиненные значения для различных свойств определенных в плане видов характеристик «Свойства объектов». Таким образом, получается набор необходимых дополнительных справочников - расширений для каждого вида свойств, использующего этот справочник в качестве типа значения.

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 9, тип кода:Строка.

Длина наименования: 50.

Не иерархический

Владельцы справочника: «Свойства объектов».

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из плана видов характеристик «Свойства объектов» и обработки «Значения свойств объектов».
