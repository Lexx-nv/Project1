Справочник «Профили изменения планов по измерениям»

Справочник предназначен для определения порядка распределения оборотов, являющихся производными от определенной величины, по элементам аналитики бюджетирования (ЦФО, проекты и т.д.) определенного вида.

Распределение производится по списку элементов соответствующего справочника; состав списка определяется при настройке профиля.

Распределение может происходить двумя способами:

  1. По фиксированным значениям коэффициентов, указанным в табличной части справочника для каждого элемента выбранной аналитики.
  2. По значениям ресурса выбранного источника данных, соответствующих элементам аналитики, выбранным в табличную часть. Соответственно, источник выбирается из списка источников данных, имеющих в составе возможных видов аналитики для распределения тот вид, для которого настраивается профиль.
